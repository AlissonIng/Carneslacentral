# Carneslacentral

## Carta de presentación

Bienvenido a **Carneslacentral**, un proyecto web desarrollado como parte del Taller de Ingeniería Web. Esta página está diseñada para presentar una carta de productos alimenticios con un enfoque visual y accesible, utilizando HTML y recursos multimedia para mostrar la oferta disponible.

El objetivo principal del proyecto es crear una experiencia sencilla y atractiva para presentar productos de carnicería y otros alimentos básicos, con una estructura clara y fácil de mantener.

## Guía principal del proyecto

### Estructura del proyecto

- `index.html`: Página principal del sitio.
- `DOCUMENTACION.md`: Documentación adicional sobre el proyecto.
- `img/`: Carpeta con imágenes utilizadas en la página.

### Cómo usar

1. Abre `index.html` en tu navegador web.
2. Revisa el contenido visual y los recursos añadidos en la página.
3. Si necesitas modificar la presentación o agregar productos nuevos, edita el archivo `index.html` y agrega las imágenes en la carpeta `img/`.

### Recomendaciones para el desarrollo

- Mantén el código HTML organizado y con comentarios cuando agregues secciones nuevas.
- Usa nombres descriptivos para las imágenes dentro de `img/`.
- Si deseas añadir estilos, crea un archivo CSS y enlázalo desde `index.html`.
